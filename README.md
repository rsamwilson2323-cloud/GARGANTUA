# 🌌 GARGANTUA

An immersive **3D black hole visualization** inspired by *Gargantua* from **Interstellar**. Built using **HTML, CSS, JavaScript, and Three.js**, the project delivers a cinematic space experience with glowing accretion disks, bloom effects, interactive camera controls, and atmospheric background audio.

---

## ✨ Features

* 🌌 Realistic black hole-inspired visualization
* ✨ Bloom and glow post-processing effects
* 🎥 Interactive camera controls with OrbitControls
* 🎵 Immersive background soundtrack
* ⚡ Smooth WebGL rendering using Three.js
* 📱 Responsive design for modern browsers
* 🚀 Lightweight and runs directly in the browser

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/rsamwilson2323-cloud/GARGANTUA.git
cd GARGANTUA
```

### Run Locally

Open `index.html` in your browser or start a local server.

Using Python:

```bash
python -m http.server 8000
```

or

```bash
npx http-server
```

Then open:

```
http://localhost:8000
```
